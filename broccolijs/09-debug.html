09-Debugging

Because Broccoli handles managing the state of the file system between transformations, it can sometimes be tricky to debug at what stage a transformation went wrong. Luckily, there is a handy and hilariously named ;) package called broccoli-stew.

The stew package contains several handy utilities, specifically the log and debug utilities.
